Zoom-bot

The Zoom-bot is an interactive navigation and maintence interface that could be implemented on subways and trains all over the world to improve riders experience. It should assist users by cleaning the train and giving them information including directions and air quality. The Zoom-bot will also keep track of train cleanliness and alert maintenance workers of problems. Subway employees will be using the interface to track trains’ cleanliness and location. The high fidelity wireframes have been developed. Click here to see the figma project.

Food Finder

My team and I designed an app for finding best-rated local food. The user is able to save notes, post them as reviews, and send them to others. We collaborated to use Zomato API on the app. I employed HTML, CSS, and JavaScript for the creation of the application. Click here to see the github repository.
